# Routing

axum provides routing utilities to conditionally run handlers based on request.
Those utilities are:

- Method routers in [`axum::routing::method_routing`]
- A path based [`axum::Router`]

[`axum::routing::method_routing`]: https://docs.rs/axum/0.6.x/axum/routing/method_routing/index.html
[`axum::Router`]: https://docs.rs/axum/0.6.x/axum/struct.Router.html
